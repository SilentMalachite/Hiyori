# 行動規範（Contributor Covenant 2.1 日本語）

このプロジェクトでは、健全で友好的かつ生産的なコラボレーションを促進するため、Contributor Covenant バージョン 2.1 に基づく行動規範を採用します。

原文（英語）: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
日本語訳（参考）: https://www.contributor-covenant.org/ja/version/2/1/code_of_conduct/

## 私たちの約束
私たちは、年齢、体型、見た目、障がい、民族性、性自認と表現、経験レベル、教育、社会・経済的地位、国籍、個人の容姿、人種、宗教、性別、性的志向、その他の特性に関わらず、コミュニティへの参加を、嫌がらせのない体験にすることを誓います。

## 私たちの基準
良い行動の例:
- 包括的で歓迎的な言葉を使う
- 異なる視点や経験に対して敬意を払う
- 建設的なフィードバックを受け入れる
- コミュニティにとって最善のことに焦点を当てる
- 他のコミュニティメンバーへの思いやりを示す

容認できない行動の例:
- 性的な言葉やイメージの使用、歓迎されない性的関心やアプローチ
- 荒らし、侮辱・中傷、個人攻撃または政治的攻撃
- 公的または私的な嫌がらせ
- 住所や電子メールアドレスなど、他者のプライベート情報の公開
- 職業上不適切と見なされるその他の行為

## 責任と権限
プロジェクトメンテナは、受け入れ可能な行動の基準を明確にし、適切かつ公正な是正措置を取る責任があります。メンテナは、問題・脅威・有害な行動と判断したコメント、コミット、コード、Wiki の編集、Issue、その他の貢献を削除・編集・拒否する権利と責任を持ちます。

## 適用範囲
この行動規範は、プロジェクト空間内および公共の場における、プロジェクトやコミュニティを代表している場合に適用されます。

## 報告
嫌がらせや容認できない行動の事例を見聞きした場合は、Issue や PR で報告してください。深刻なケースは GitHub の報告機能や、必要に応じて GitHub Support へご連絡ください。

## 執行
行動規範の違反は、メンテナの裁量により対応されます。是正措置の例（警告、一時的な参加停止、恒久的な追放など）については、Contributor Covenant 2.1 の「執行指針」を参照してください。

## 帰属
